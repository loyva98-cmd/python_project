#Реализация модуля для анализа рядов и последовательностей на сходимость

Проект выполняется самостоятельно Михайловым Львом Александровичем

Общая идея: Реализовать различные критерии, условия и признаки сходимости рядов и последовательностей(возможно ещё добавить аналоги для интегралов равно как и для функциональных последовательностей) в качестве функций модуля

Цели на итерации(1 и 2):

I. 1. Проверка сходимости по определению
2. Реализация проверки на сходимость по критерию Коши рядов и последовательностей
3. Признак Дирихле сходимости числовых рядов

II. 1. Признак Даламбера
2. Признак Абеля сходимости числовых рядов
3. Признак Вейерштрасса равномерной сходимости функциональных последовательностей
